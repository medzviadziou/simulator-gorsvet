# Simulator Gorsvet

Симулятор логики управления городским освещением:
- ручной и автоматический режимы;
- локальное управление каналами `KM1..KM3`;
- SCADA-команды;
- имитация аварий и задержек индикации.

## Запуск

```bash
npm install
npm run dev
```

## Справка

- Встроенная краткая справка: `public/help.html`
- В интерфейсе приложения доступна кнопка **«Справка»** (в правом верхнем углу).
